把本地修改同步回远程 Git 仓库（保留你的修改）
如果你的修改满意，想同步到远程 Git 仓库（防止本地文件丢失，后续也能在其他地方获取），步骤：
VS Code 终端依次输入 3 个命令
第一步：把所有修改的文件加入暂存（.表示所有文件）
bash
运行
git add .
第二步：提交修改，填写修改说明（引号里的内容可以自定义，比如「微调 UI，修改按钮圆角和主色调」）
bash
运行
git commit -m "微调UI样式，优化按钮和图片展示效果"
第三步：推送到远程 Git 仓库
bash
运行
git push
登录验证（首次推送需要）
如果是 GitHub，可能会提示输入「个人访问令牌」（不是密码），需要在 GitHub 上提前创建；如果是 Gitee，直接输入账号密码即可。
推送成功后，刷新你的远程 Git 仓库，会发现文件已经更新为你本地修改后的版本。
